## Icon 图标

### 使用指南
``` javascript
import { Loading } from 'afo';

Vue.use(Loading);
```

### 代码演示

#### 基础用法

设置`type`属性为对应的图标名称即可

```html
<afo-loading size=40 />
```

### API

| 参数 | 说明 | 类型 | 默认值 |
|-----------|-----------|-----------|-------------|
| size | 图标名称 |`Number` | 30 |
| color | 图标颜色 | `String` | `black|white` |
